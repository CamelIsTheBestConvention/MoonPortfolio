💡 프로젝트 소개

학생들끼리 문제풀이할 때 서로 본인들의 답을 공유해보면 어떨까 생각하여 만들게 되었습니다.

페이지 주요 기능

  • 회원가입, 로그인
  • 검색 기능
  • 마이 페이지
  • 게시판 글 작성
  • 댓글 작성
  • 포인트 충전

- MVC2패턴을 사용해 오라클과 연동

- 인텔리제이를 사용해 각 코드에 맞는 어노테이션으로 구성

- 화면에서는 타임리프를 사용해 데이터를 요청

- AJAX를 사용한 비동기방식의 게시판 구성

맡은 부분

- 프론트엔드 : 로그인, 회원가입, 관리자 페이지, 글쓰기 페이지 퍼블리싱

- 백엔드 : 게시판 전체, 포인트 충전 파트

개발 과정

- 웹 개발 강의의 스프링 부분 강의를 공부하며 팀원들과 같이 작업한 첫 스프링 프로젝트입니다.

- 처음에 AJAX의 요청 방식을 이해하기 어려워 적용까지 시간이 오래 걸렸지만, 현재는 원하는 곳에 사용가능합니다.

- 백엔드를 작업할 때 본인이 퍼블리싱한 페이지를 맡는게 아니였기 때문에 서로 소통과 코드리뷰를 잘 했어야 했는데 이 부분이 많이 어려웠습니다.

- 그래도 서로 부족한 부분을 보완해주며 꾸준히 작업한 결과 기본적인 틀은 갖출 수 있게 되었습니다.

- 발표 당일날 못 만든 부분은 강의가 끝나고 하나씩 수정하며 고치는 중 입니다.

- 포인트 충전은 카카오톡 qr결제를 이용하여 실제 결제가 되는 것이 아니라 api로 가상으로 결제하게 만들었습니다.

개선점

- 원래 게시판에 첨부파일까지 등록 가능하게 하고 싶었으나, 학습이 부족해 첨부파일이 빠져서 보완해야 합니다.

- 처음에 계정삭제를 관리자페이지에서 맡아버리는 기획오류를 저질러 마이페이지에서 탈퇴 가능하게 변경해야 합니다.

- 코드가 많이 복잡해 페이징 처리가 잘 안되어 쿼리문을 재수정 해야 합니다.

- 메인 페이지에 각 과목 카테고리로 게시판 이동하는 버튼이 있는데, 게시판이 AJAX로 구성되어 현재 전체 글로만 보내지기 때문에 수정해야 합니다.
